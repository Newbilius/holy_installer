# Holy Installer 1.0

Небольшая утилитка, предназначенная для создания патчей структуры SQL базы данных, а так же - развертывания/создания бэкапа базы.

## Что умеет:

* Сохранение структуры выбранных таблиц в XML файл-патч. В файл попадет список таблиц, список столбцов в них, типы столбцов.
* Установка полученного патча
* Экспорт базы данных в файл в специфичном формате "одна строка - одна SQL-команда"
* Импорт базы из такого файла

## Чего **пока** не умеет (будет в следующем релизе)

* Сохранять и восстанавливать из патча сложные настройки (внешние и уникальные ключи, комментарии к столбцам)

## Как устанавливать

1. Распаковать в любую папку
2. Скопировать файл options/options_demo.php в options/options.php и поправить под свой проект.

Изначально написанная для моей системы HolyCMS, а сейчас - вынесена в отдельный проект.
